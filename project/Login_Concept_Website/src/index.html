What website is this?

    This is a website for testing login and account registration. The goal is to understand how to create a login and registration system. All accounts registered are stored in a sessionStorage.

Note: Do not use your real email when registering on this website!

Javascript sessionStorage technology

How this website works

    Starting from the registration page. On the registration page, there are 2 forms that can be filled out: Name and Password, and there is a register button. The register button when pressed will trigger a Javascript event and manage the values that have been entered in both input forms. The form containing the username will be made the value in the sessionStorage, while the form containing the password will be made the keyItem in the sessionStorage. If the value is empty, it will fail to execute and display an invalid message.

    Next is the login page. Once again there are 2 forms, Name and Password, and one login button. When the login button is pressed, it will trigger a Javascript event to manage both forms. The values contained in the password and name will be matched to see if they exist in the sessionStorage; if the keyItem (password) and value (name) match, then the page will be redirected back to the main menu and the guest name will be changed to the one you entered. If it does not match, it will return a failure message and require the user to fill in the login correctly again.
