Apa itu CSS ?

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman. Saya menggunakan css untuk desain tampilan website agar nyaman untuk di lihat, karena suatu desain yang bagus juga menentukan suatu karya bagus atau tidak.
