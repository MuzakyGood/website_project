CSS Universal Selector

What is Universal Selector ?

The universal selector is used to apply to all elements. However, this selector can also specifically target a single element by combining it with other selectors. The universal selector in CSS starts with an asterisk (*) before writing methods such as class, id, or attribute.

Selektor universal digunakan untuk diterapkan pada semua elemen. Namun, selektor ini juga dapat secara spesifik menargetkan satu elemen dengan menggabungkannya dengan selektor lain. Selektor universal dalam CSS dimulai dengan tanda bintang (*) sebelum menulis metode seperti class, id, atau attribute.
