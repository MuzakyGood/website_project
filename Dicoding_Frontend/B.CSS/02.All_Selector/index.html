Type Selector

The type selector only works on the targeted element. Here I am targeting the span element.

Paragraph elements are unaffected.

Class Selector

The class selector must include the class attribute on the HTML element, and in CSS it must use a dot (.) before the class name.

Id Selector

The ID selector on an HTML element must include the id attribute, and in CSS it must include hashtag (#) before the id name.
