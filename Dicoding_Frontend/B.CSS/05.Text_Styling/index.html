Line Height

The line-height property is used to set the minimum distance from the baseline of the text to the line when displaying it on a web page.

Text Indent

The text-indent property is used to create a space/indent at the beginning of a paragraph. We can specify the value of this property using length measurements in px, em, and in, or we can use a percentage %. The percentage value is calculated based on the width of the parent element.
