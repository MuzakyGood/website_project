Pseudo-element Selector

  • ::before and ::after

    This selector is commonly used when we want to add content immediately before and after a paragraph element. We do not need to write new elements/content in the HTML file to add content. However, we can make use of pseudo-elements to do so.

  • ::first-letter

    Another one is ::first_letter, which is a pseudo-class in CSS, used to style just the first letter in a paragraph.
