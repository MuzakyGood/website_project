Descendant Selector (space)

The Descendant Selector is similar to the child selector, but its hierarchy is broader because the rule will be applied to all elements that are its descendants, even indirectly. The first basic selector written in this selector becomes the parent, and the second basic selector will apply the rule. This selector uses a space to combine the two basic selectors.
